Pointwise Glyph Script Exchange

Macros and Templates for CFD Meshing

Welcome to the Pointwise Script Exchange

Glyph, the Tcl-based scripting language in Pointwise and Gridgen, offers the ability to automate all or part of the CFD meshing process.

  • Create macros that automate tedious tasks.
  • Write a template to produce geometry or generate an entire mesh.
  • Build libraries of custom tools.
  • Just have fun making the software do something it wasn't designed to do.

Available Glyph Scripts

Below are links to all available Glyph scripts. If you are a GitHub user, subscribe to your favorite scripts to receive updates on development progress or new version information. Fork scripts and modify the code to create new projects. Who knows, we may merge your changes back into our master branch. Enjoy!

Script Name Description
AirfoilGenerator Generate NACA 4-series airfoil geometries
AirfoilMesh Generate airfoil geometries and boundary layer meshes
ArrayCopyRotate Copy, rotate, and paste entities a prescribed number of times

Authors and Contributors

We welcome new script authors and contributors! Either submit a new script or fork an existing project on GitHub. Please be sure to include sample files so we can experiment with the script before posting or merging your code.

Don't have a GitHub account, that's alright. Just email us your script. Please include the following information in your email:

  • Your name
  • The name of your organization
  • Your email address
  • A description of the script's purpose
  • The script itself and any related files included as attachments

Thanks again. We hope to review and post your script shortly.
